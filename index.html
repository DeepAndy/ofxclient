Ofxclient

Bank transaction downloader and python OFX client libraries

Download this project as a .zip file Download this project as a tar.gz file

Overview

Simple OFX client libraries.

Note: the web application that used to be here has now been moved to another project.

Installing

OSX

$ sudo easy_install ofxclient

Windows

$ easy_install ofxclient

Note: Make sure %PYTHON_HOME%\Scripts is added to your %PATH% otherwise the 'ofxclient' command will not be found on your path.

Source

get source from https://github.com/captin411/ofxclient unpack it

$ sudo python setup.py install
